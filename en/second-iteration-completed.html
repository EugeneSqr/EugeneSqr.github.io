Second Iteration Completed

2014-08-13

New features

I've just finished the second iteration of building my personal web site.

  • dates localization
  • post listing paging
  • server side storage optimization
  • google analytics
  • default language changed to Russian

Postponed tasks

  • old articles migration, because I decided to focus on new content and leave my old blog without changes
  • automated spell checking, because the number of posts I write is relatively small; I should be able to handle spell checking with third party tools manually.

New iteration scope

The site is fully functional and I like the way I can share my thoughts using it. Social features are still missing which is sad, but I'm ok with that (at least for some period of time). What is more important is that I finally got these books from Amazon:

  • Effective C++: 55 Specific Ways to Improve Your Programs and Designs
  • Effective STL: 50 Specific Ways to Improve Your Use of the Standard Template Library

so I'm going to spend some time studying them.

Most likely my next post will be something around C++ :)
